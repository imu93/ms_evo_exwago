# Code and files related to: Comparative transcriptomics revealed exWAGO as an ancient regulator of LTR retrotransposons
---
## ANNOTATIONS
GFF3 files of segmented genome annotations including the doble strand non-overalping regions of:
1. Portein-coding genes
2. Curated repetitive elements
3. Non-coding RNA families
4. Intergeneic regions
---

## CONTROL_LIBRARIES
FASTA files with control repeat libraries (Improving transposable element annotation in Strongylida genomes). 
---

## COUNT_TABLES
Tab separated txt files of sRNA counts (18-27 nt length) produced from the resulting bam files of ShortStack, and featureCounts.
---

## DE_TABLES
Files containing the results of diffrential expression analysis with edgeR, and rRNA_S based normalization factors.
---

## LM_TABLES
Tables used for linear models (ExWAGO is an ancestral regulator of LTR retrotransposons, and Structural differences among LTRs drive their sRNA secretion pathway).
---

## REPEAT_LIBRARIES
FASTA files with the repeat libraries.
> Note: Only ancylostoma_ceylanicum.repeats.fa, caenorhabditis_elegans.repeats.fa, heligmosomoides_bakeri.repeats.fa, heligmosomoides_polygyrus.repeats.fa, nippostrongylus_brasiliensis.repeats.fa, and teladorsagia_circumcincta.repeats.fa, can be considered as manualy curated. The rest of libraries were automaticaly built with earlGrey.
---

## src
Code used for annotation, diffrential expression, enrichement, and diversity analyses.
